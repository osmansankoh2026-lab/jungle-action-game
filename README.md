# Jungle Action Game

Complex jungle action game with AI teammates, smart enemies, NPC rescue mechanics, and real-time combat.

## Features

- AI-controlled team members
- Smart enemy AI
- NPC rescue system
- Real-time combat
- Loot system
- Mobile controls

## Installation

```
Google play store install
Google android
```

## Game Mechanics

- Rescue 3 NPCs per level
- Defeat all enemies
- Manage team members
- Collect weapons and armor
- Survive combat
